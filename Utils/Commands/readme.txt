Все новые команды добавляются в пространство имен Utils.Commands, 
наследуются от Command и должны начинаться с префикса Com

Добавлять новую группу команд лучше новым файлом, не трогая старые файлы

Как добавлять:
1. Реализуем абстрактные GetByteData(первым должен кодироваться int base.type) и SetFields(здесь без base.type)
2. Добавляем конструктор без аргументов(зачем - ниже)
3. Добавляем руками команду в базовом классе в Command.COMMAND_LIST
4. Обязательная часть кончилась, остальные методы\поля\классы - по желанию

Как происходит восстановление команды после ее передачи по сети: 
через Command.CreateConcrete вызывается пустой конструктор, затем SetFields.

За чем нужно следить:
При создании команды на сервере нужно присвоить base.clientID - какому клиенту слать это сообщение
